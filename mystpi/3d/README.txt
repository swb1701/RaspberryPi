# Summary

I'm a big fan of the Myst / Riven / Uru worlds and I wanted to model one of the many lanterns scattered throughout the ages, but figured the Stone Ship Age lighthouse had good potential. Also the geometry is very simple so it was  super easy to model.

I intend to add to this model a support for an LED candle light inside so it can be used as a table top lantern. I will publish any relevant additional .STLs when I do.

At scale, this model is 30 cm (12 inches) tall.
